# 📡 Telecommunications

Welcome to the **Telecommunications** section of my digital garden. This folder contains detailed notes of the topics listed below 

---

## 📚 Topics Covered

The material is organized following a logical path from signal theory to modern modulation techniques:

1.  **Signals and Spectra:** Fourier Analysis, frequency domain representation.
2.  **LTI Systems:** Linear Time-Invariant systems, convolution, and impulse response.
3.  **Filters:** Ideal and real filters, bandwidth, and frequency response.
4.  **Signal Properties:** Energy, Power, and Periodic signals analysis.
5.  **Sampling:** Nyquist-Shannon theorem, aliasing, and reconstruction.
6.  **PCM (Pulse Code Modulation):** Quantization, encoding, and bit rate.
7.  **PAM (Pulse Amplitude Modulation):** Baseband transmission basics.
8.  **Multiplexing (TDM & FDM):** Time and Frequency Division Multiplexing techniques.
